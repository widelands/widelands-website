Webchat

This is a webchat applet provided by the freenode network. This is a convenient solution if you just want to chat in the #widelands channel on freenode.

If you are not familiar with IRC (Internet Relay Chat), here's a few tips:

  • Please be patient. #widelands is fairly low traffic so it might take a while before someone sees your question and replies.
  • If you wish to change your nickname, do this with the command /nick. For instance type /nick wl in order to change it to wl.
  • We also have a german speaking IRC channel. To join it, simply type /join #widelands-de
